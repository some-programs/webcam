# Main view key bindings

| Key        | Function                   |
|------------|----------------------------|
| ?          | Toggle help view           |
| F1-F12     | Load preset 1-12           |
| ALT+F1-F12 | Save preset 1-12           |
| /          | Filter controls            |
| `-1-0      | Quick set value            |
| →   l      | Value increase             |
| ←   h      | Value decrease             |
| y          | Single step value decrease |
| o          | Single step value increase |
